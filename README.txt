Для запуска приложения: 
1)Пройдите по директориям out/artifacts/airpotr_search_jar(в данному пути так же будет находиться файл airports.csv, для корректной работы программы не извлекайте его)
2)Откройте консоль в этой папке(можно нажать левый shift + правая кнопка мыши, после этого нажмите пункт "Открыть окно PowerShell здесь")
3)Введите команду java -jar airport-search.jar <номер столбца>
4)Следуйте инструкциям программы

Исходный код находится в папке src
